tags semanticas para texto

HTML, ou HyperText Markup Language, é a linguagem de marcação padrão usada para criar páginas da web. Ela fornece a estrutura básica de um documento, utilizando elementos que definem cabeçalhos, parágrafos, links, imagens, tabelas e outros conteúdos. HTML usa tags (como <p p >, < a > ; entre outras) para indicar a função de cada parte do conteúdo. É fundamental para o desenvolvimento web, pois permite que navegadores interpretem e exibam o conteúdo corretamente.
